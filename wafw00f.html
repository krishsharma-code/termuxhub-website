WAFW00F: Web Application Firewall Detector

Introduction to WAFW00F

WAFW00F is a powerful reconnaissance tool designed to identify and fingerprint Web Application Firewalls (WAFs) protecting a website. It sends a series of HTTP requests and analyzes the responses to detect unique signatures left by various WAF vendors like Cloudflare, AWS WAF, and ModSecurity.

Installation Commands

1. Update Environment

pkg update && pkg upgrade

2. Install Python & Git

pkg install python git

3. Install WAFW00F via Pip

pip install wafw00f
